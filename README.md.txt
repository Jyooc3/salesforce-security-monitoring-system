# Salesforce Security Monitoring System

## Project Overview
This project demonstrates a security monitoring system built using Salesforce.

It detects high-risk customer records and automatically sends alerts to administrators using Flow Builder.

## Features
- Custom Object: Customer Security Record
- Custom Fields: Email, Phone, Credit Score, Risk Level
- Field-Level Security to protect sensitive data
- Record Triggered Flow for automation
- High-risk detection system

## Flow Logic
Customer Record Created
        ↓
Risk Level evaluated
        ↓
If Risk Level = High
        ↓
Flow triggers alert email

## Technologies Used
- Salesforce
- Flow Builder
- Object Manager
- Field-Level Security

## Purpose
This project demonstrates Salesforce Admin skills including configuration, security controls, and automation.
